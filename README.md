# ASR Terminal

A polished, interactive terminal-themed portfolio website for Aditya Surendra Rekhe, built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and React Icons.

## Features

- Animated boot sequence and terminal-style interface
- Interactive commands such as help, about, skills, experience, projects, education, achievements, leadership, workshops, contact, socials, resume, neofetch, matrix, ascii, theme, quote, fortune, coffee, hack, sudo, uptime, and more
- Expandable project modal cards
- Responsive layout for desktop and mobile
- SEO metadata, favicon, manifest, robots.txt, and sitemap for deployment readiness

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is production-ready for Vercel and can be deployed directly from the repository root.

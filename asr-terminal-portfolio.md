A Linux terminal-inspired portfolio website built to showcase my projects, experience, and journey as a software engineer.

Unlike traditional portfolio websites, this one is designed to feel like you're SSH'ing into my personal machine. Everything is command-driven, keyboard-friendly, and inspired by Unix terminals.

## Why a Terminal?

Because developers spend more time in terminals than browsers.

Instead of scrolling through endless sections, visitors can interact with my portfolio just like a real Linux shell.

```
visitor@portfolio:~$ help

Available commands:

about
skills
experience
projects
education
achievements
workshops
contact
socials
resume
clear
help
theme
matrix
ascii
exit
```

The goal wasn't to build another portfolio.

The goal was to build something people remember.
